Необходимо разработать сайт для удобного просмотра актуальных бонусных баллов студентами.

## Release 0

Для начала в приложении тебе хватит 2х таблиц: одна на пользователя, а другая на студента. Один пользователь может иметь много студентов, а студент принадлежит пользователю. У студента точно есть поля - `name, bonus`

## Release 8*

Почти всегда сайты в интернете показывают информационное модальное окно, что о пользователе собирается персональная информация в Cookie. Добавь такое окно и в своё приложение:

- При первой загрузке страницы пользователю (даже неавторизованному) показывается информационное сообщение
   - Если пользователь согласился с использованием Cookie, то данное сообщение уже никогда не будет показываться этому пользователю (не важно, авторизован он или нет)
   - Если пользователь закрыл информационное сообщение, то оно должно показаться и в следующий раз при загрузке новой страницы
- Само окно не должно предотвращать возможность взаимодействия с твоим приложением. Другими словами, пользователь должен иметь возможность регистрироваться, добавлять и удалять студентов даже с открытым окном "Мы храним информацию в Cookie".
